# Domain: newb (뉴비 훈련소)

`dnfm.kr` / `www.dnfm.kr` — 던파 모바일 뉴비 훈련소 카톡방 공식 웹.

## 1. 본업

**카톡방 "뉴비 훈련소" 의 운영 허브**. 운영자(=사용자, 방장) 가 본인이 쓰는 공식 웹.

세 가지 역할이 한 사이트에 모임:
1. **공식 포털 연결** — 던파 모바일 공식(nexon dnfm) 으로 가는 단일 게이트. 공지/이벤트/추천 가이드 등.
2. **카톡방 운영 보조** — 신규 모험가 가이드, 오늘 할 일 체크리스트, 자주 묻는 질문 회수, 톡방 입장 동선.
3. **봇/이벤트 개발의 뿌리** — 향후 카톡방 챗봇, 이벤트 페이지, 자동 알림 등이 이 사이트의 데이터·인증을 기반으로 확장됨.

## 2. 운영 주체

- **사용자(=방장=본 레포 개발자) 본인**. 비개발자 UX 강도 약함 — 콘텐츠 수정·기능 추가 시 코드/DB 직접 만지는 게 OK.
- 톡방 사용자(뉴비 모험가) 는 사이트의 **소비자** 이지 운영자가 아님. 일반 회원 가입은 [[shared]] 정책 따라 가능하나, newb 자체 어드민 UI 의 self-service 도구 우선순위는 낮음.

## 3. 콘텐츠 SSOT

- 정적 콘텐츠 (hero/가이드 카드/체크리스트/링크 그룹): `newb/src/lib/content.js` — 코드 수정 + git push.
- 동적 콘텐츠 (공지 글, 이벤트, 봇 데이터, 사용자 글): Stage 2 이후 `api.dnfm.kr` backend.
- 운영 자료 위치 지침: `docs/content-operations.md`.

## 4. 디자인 톤

- 공식 던파 모바일 홈 톤에서 영감 (라이트 톤, 게시판 허브 레이아웃, gold/red 액센트).
- **공식 wordmark / 공식 게시판 카테고리 directly copy 금지** — [[shared]] 의 TM 회피 정책.
- 자체 브랜드 = `DNFM.KR / 던파 모바일 뉴비 훈련소`. 본문 메뉴/CTA 는 실재 섹션 기반.

## 5. 초기 백로그 (사용자 발언 기반)

- **카톡방 챗봇 연동** — newb 사이트 인증/회원/이벤트 데이터를 봇이 호출. 챗봇 spec 은 미정.
- **이벤트 페이지** — 친구 사이트(allow) 의 콘테스트와 별개로 newb 자체 이벤트도 운영 가능.
- 콘텐츠 동적화 (공지/이벤트 게시판) 는 사용자 본인이 코드로 직접 추가 — UI self-service 우선순위 X.
