# 콘텐츠 운영 가이드

## 빠르게 바꿀 곳

각 사이트의 화면 문구와 링크는 해당 앱의 `src/lib/content.js`에 있습니다.

- 카톡방 URL / 공식 링크 / 가이드 카드 / 체크리스트: `src/lib/content.js`
- 친구들 `allow.dnfm.kr` 콘텐츠는 **별도 repo** (`dnfm-allow`) 에서 관리. 이 repo 와 무관.

## 링크 원칙

확정되지 않은 링크는 `url: null`과 `reason`을 함께 둡니다. 그러면 화면에서는 비활성 버튼으로 표시되어 방문자가 잘못된 링크를 누르지 않습니다.

```js
{
  label: "카톡방 입장",
  url: null,
  reason: "오픈채팅 URL 등록 전"
}
```

실제 URL이 생기면 아래처럼 바꿉니다.

```js
{
  label: "카톡방 입장",
  url: "https://open.kakao.com/o/example"
}
```

## 공식 링크 확인 기준

외부 공식 링크는 배포 전 한 번씩 브라우저에서 열어 확인합니다. 던파모바일 공식 사이트 메뉴 구조는 변경될 수 있으므로, 공지/이벤트/추천 가이드 링크가 404나 로그인 페이지로 바뀌면 `src/content.js`를 먼저 갱신합니다.
