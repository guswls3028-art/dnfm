# 프로젝트 브리프

## 목적

던파 모바일 뉴비 훈련소 카카오톡방을 운영하기 위한 정보 허브를 만든다. 신규 유저가 카톡방, 공식 홈페이지, 공지사항, 이벤트, 추천 가이드, 운영 문의로 빠르게 이동할 수 있어야 한다.

## 확장 목적

`dnfm.kr` 도메인을 중심으로 하되, 스트리머나 협력자에게 하위 사이트를 제공할 수 있게 한다. 첫 대상은 허락님이며 `allow.dnfm.kr`로 연결한다. 각 사이트는 독립 Next.js 앱으로 분리한다.

## 초기 범위

- Next.js 앱 2개
- `newb/`: 뉴비 훈련소, `dnfm.kr`
- `allow/`: 허락님 페이지, `allow.dnfm.kr`
- 공식 링크 묶음
- 운영 체크리스트
- 가이드 카드와 필터
- 허락님 스트리머 페이지 템플릿

## 아직 필요한 실제 데이터

- 카카오톡 오픈채팅 URL
- 허락님 라이브 채널 URL
- 허락님 유튜브 또는 다시보기 URL
- 운영 문의 메일의 실제 수신 설정
- 자체 가이드 문서 주소
